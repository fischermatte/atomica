The Game

Ziel des Spiels ist es, farbige Kreise (Atome) auf dem Spielbrett zu bewegen und mit vier oder mehr gleichfarbigen Atomen einen rechteckigen Block (Molekül) zu bilden. Das Spiel läuft rundenweise ab und in jeder Runde dürfen Sie (mindestens) ein Atom bewegen.


Wird mit einem Zug ein Molekül gebildet, verschwinden die beteiligten Atome daraufhin vom Spielbrett und Sie erhalten eine Punktzahl, entsprechend der Anzahl der Atome aus denen das Molekül bestand.


Das Spielbrett ist rechteckig und besteht aus ebenfalls rechteckigen Feldern, die in einem Schachbrettmuster angeordnet sind. Die Atome können beliebig viele Felder weit auf dem Spielbrett bewegt werden, sofern der Weg nicht durch ein anderes Atom blockiert ist. Auf dem Spielbrett erscheinen zu Beginn jeder Runde an zuf¨alligen Positionen farbige Icons (im Folgenden Indikator-Icons genannt). Diese Icons markieren die Felder, auf denen zu Beginn der darauf folgenden Runde neue Atome entstehen.


Das Spiel besteht aus mehreren Leveln. Um ein Level zu gewinnen, müssen Sie eine vorgegebene Anzahl an Punkten erlangen, indem Sie mit den Atomen Moleküle bauen. Das Spiel ist verloren, sobald das komplette Spielbrett bis auf ein Feld mit Atomen gefüllt ist und kein Atom mehr bewegt bzw. nur noch ein einziges Atom zwischen zwei angrenzenden Feldern hin und her bewegt werden kann.


Controls

Wählen Sie das zu verschiebene Atom mit der linken Maustaste aus. Das Atom bleibt so lange zum Verschieben aktiviert bis Sie ein anderes Atom auswählen. Nachdem Sie ein freies Feld ausgewählt haben, können Sie durch erneutes Klicken der linken Maustaste das Atom an die gewünschte Stelle verschieben.


Load Game

Läd ein gespeichertes Spiel. Ein Spiel kann während des Spielverlaufs oder über den Spiel Editor gespeichert werden.


Game Editor

Mit dem Spiel Editor können Sie eine initiale Spielsituation selber erstellen. Sie haben dabei die Auswahl zwischen 3 (entspricht Level 1) und 12 (entspricht Level 10) Farben. Die Atome und Indikatoren können beliebig plaziert werden, es dürfen allerdings dabei keine Moleküle entstehen. Sie dürfen maximal 3 Indikatoren auf dem Spielbrett platzieren.


Options

Ermöglicht das Ändern der Spieleinstellungen.

Basispunkte

Über den Basispunktewert kann die Höhe der Punktevergabe beim Bilden eines Molecüls beeinflusst werden. Sei f die Anzahl der möglichen Farben im aktuellen Level. Dann berechnet sich die Anzahl p4 der Punkte, die der Spieler für jedes Molekül aus vier gleichfarbigen Atomen erhält, für das aktuelle Level wie folgt:

p4 := (f − 2) * b

Spielbrettgröße

Es können die Anzahl der Zeilen und Spalten des Spielbrettes angegeben werden (zwischen 3 und 30)).

Max. Punkte

Für jedes Level kann die Höhe der Punkte angegeben werden, welche benötigt werden um in das nächst höhere Level zu gelangen.


Language

Ermöglicht die Auswahl einer Sprache. Es kann zwischen English und Deutsch gewählt werden.
